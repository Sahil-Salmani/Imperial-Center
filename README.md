# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](https://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and [`typescript-eslint`](https://typescript-eslint.io) in your project.

# Imperial-Center
Imperial Center is a modern luxury mall website designed to provide users with a premium digital shopping and entertainment experience. Inspired by world-class destinations such as Dubai Mall and modern luxury brand websites, the platform combines elegant design, immersive visuals, smooth animations, and responsive layouts to create a sophisticated online presence.

The website showcases a wide variety of luxury shopping brands, fashion avenues, premium dining experiences, entertainment attractions, and exclusive lifestyle services. Visitors can explore modern retail collections, discover fine dining restaurants and cafés, experience cinematic entertainment sections, and stay updated with the latest offers and events happening at the mall.

Imperial Center focuses heavily on user experience by integrating interactive features such as autoplay image carousels, video hero sections with custom controls, hover animations, responsive navigation menus, and visually engaging layouts. The website is fully responsive, ensuring seamless performance across desktop, tablet, and mobile devices.

Developed using React.js, Tailwind CSS, and Swiper.js, the project demonstrates modern frontend development practices and clean component-based architecture. The design emphasizes luxury aesthetics through elegant typography, fullscreen imagery, dark overlays, smooth transitions, and interactive UI elements.

Overall, Imperial Center is more than just a shopping mall website — it is a complete digital lifestyle destination that reflects luxury, innovation, entertainment, and modern web design in a visually immersive way.
